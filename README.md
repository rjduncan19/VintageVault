# VintageVault

Cloud-to-cloud backup for families, freelancers, and small businesses. Protect your files from ransomware, accidental deletion, and account compromise by backing up to a second cloud provider.

---

## Development Setup

VintageVault can be developed in three complementary environments:

- **Local VS Code** — Recommended for long development sessions and full IDE features
- **GitHub Codespaces** — Cloud IDE for quick edits and cross-machine access
- **Copilot CLI** — Terminal-based AI-assisted development

All three environments sync via Git, so you can switch between them seamlessly.

**See [CONTRIBUTING.md](CONTRIBUTING.md) for complete setup instructions.**
